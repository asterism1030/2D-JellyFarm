# 2D-JellyFarm

설명 :   
- Unity 환경에서의 2D 게임을 개발해보고자 시작했습니다
- FSM 패턴을 사용하여 젤리의 동작을 구현하였습니다
- 젤리를 해금하여 구매하고 젤리를 터치하여 재화를 얻습니다

분류 :   
클리커 방치형 게임

환경 :   
Unity 2021.3.12f1
